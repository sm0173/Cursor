# Источники для главы 1 и списка литературы

## Что хранить здесь

- Выгрузки страниц сайта организации, фрагменты годовых отчётов (если разрешено).
- PDF статей и книг (или ссылки с сохранёнными библиографическими данными).
- Файл `библиография.bib` или таблица — по желанию; основной список оформляется в Word по ГОСТ.

## Минимум для оценки по методичке

Не менее **20** источников, в том числе **англоязычных**. Каждый факт из главы 1 должен быть привязан к источнику.

См. также: [библиография_шаблон_ГОСТ.md](библиография_шаблон_ГОСТ.md).
