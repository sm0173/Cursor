# Курсовой проект «Моделирование организации»

Рабочая папка для отчёта по методическим указаниям ВШЭ (2-е изд., 2022). Здесь хранятся источники, эмпирика, экспорты моделей и тексты глав.

## Структура

| Папка / файл | Назначение |
|--------------|------------|
| [КОНТЕКСТ_ПРОЕКТА.md](КОНТЕКСТ_ПРОЕКТА.md) | Объект исследования, границы процесса, допущения — **заполнить в первую очередь** |
| [ЧЕКЛИСТ_методичка_ВШЭ.md](ЧЕКЛИСТ_методичка_ВШЭ.md) | Соответствие обязательным требованиям |
| [ПРАВИЛА_ДЛЯ_АГЕНТА.md](ПРАВИЛА_ДЛЯ_АГЕНТА.md) | Как давать контекст ИИ в Cursor без галлюцинаций |
| `01_источники/` | PDF, ссылки, выдержки из литературы и официальных материалов |
| `02_эмпирика_сырьё/` | Заметки интервью (анонимизированные), опросы, сырые данные |
| `03_модели_экспорт/` | PNG/SVG/PDF из ARIS, bpmn.io, Bizagi и т.п. + версии |
| `04_текст_главы/` | Черновики и финал глав, введение, заключение |
| `05_презентация/` | Структура и материалы к защите (12–15 слайдов, 10 мин); [чеклист перед сдачей](05_презентация/чеклист_оригинальность_и_сдача.md) |

## Быстрый старт

1. Заполните **КОНТЕКСТ_ПРОЕКТА.md**.
2. Идите по **ЧЕКЛИСТ_методичка_ВШЭ.md** сверху вниз.
3. В каждой главе начинайте с шаблона в `04_текст_главы/`.

Оригинальность текста консультационного проекта — **не ниже 80%** (п. 3.3 методички).
