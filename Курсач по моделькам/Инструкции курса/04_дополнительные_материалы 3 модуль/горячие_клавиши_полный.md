# Все горячие клавиши Cursor — полный справочник

> Mac-first. Windows/Linux в скобках.

---

## 🤖 AI-функции (самое важное)

| Действие | Mac | Windows |
|----------|-----|---------|
| AI Chat (боковая панель) | `Cmd+L` | `Ctrl+L` |
| Добавить выделенное в Chat | `Cmd+Shift+L` | `Ctrl+Shift+L` |
| Composer (мультифайловый) | `Cmd+I` | `Ctrl+I` |
| Inline Edit (быстрая правка) | `Cmd+K` | `Ctrl+K` |
| Принять Inline Edit | `Cmd+Shift+K` | `Ctrl+Shift+K` |
| Tab (принять автодополнение) | `Tab` | `Tab` |
| Отклонить автодополнение | `Esc` | `Esc` |
| Следующее предложение | `Alt+]` | `Alt+]` |
| Новый чат | `Cmd+N` (в панели чата) | `Ctrl+N` |

## 💡 Топ-5 для начинающих

Если запоминаешь только 5 комбинаций — запомни эти:

1. `Cmd+L` — AI Chat (задать вопрос)
2. `Cmd+I` — Composer (создать/изменить файлы)
3. `Cmd+K` — быстрая правка выделенного
4. `Cmd+P` — открыть любой файл по имени
5. `Cmd+S` — сохранить (очевидно, но забывают)
