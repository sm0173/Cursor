# Claude Code — система агентов

## Что такое Claude Code

Claude Code — это AI-агент от Anthropic, который работает в терминале. В отличие от Cursor (где ты пишешь промпты в GUI), Claude Code — это полноценный агент: он сам читает файлы, пишет код, запускает команды и исправляет ошибки.

Для бизнес-ассистента это интересно как продвинутый уровень: когда ты перерос Cursor и хочешь автоматизировать сложные задачи.

## Структура проекта в Claude Code

```
мой-проект/
├── CLAUDE.md          ← Главный файл инструкций (как .cursorrules)
├── .claude/
│   ├── settings.json  ← Настройки проекта
│   └── agents/        ← Папка с агентами
│       ├── reviewer.md
│       ├── writer.md
│       └── analyst.md
└── ... (файлы проекта)
```

### CLAUDE.md — «мозг» агента

Аналог `.cursorrules` в Cursor. Claude Code читает его автоматически при старте. Пишешь сюда:
- Кто ты и что за проект
- Стиль общения
- Правила работы
- Какие инструменты использовать

### .claude/agents/ — команда агентов

Каждый файл — отдельный агент со своей ролью. Как `.mdc` файлы в Cursor, только для Claude Code.

Пример `reviewer.md`:
```markdown
# Ревьюер кода

## Роль
Проверяешь код на ошибки, безопасность и читаемость.

## Формат
1. Критические проблемы (блокеры)
2. Предупреждения
3. Рекомендации по улучшению
```

## Чем отличается от Cursor

| | Cursor | Claude Code |
|---|---|---|
| Интерфейс | GUI (окно) | Терминал |
| Запуск | Открыл редактор | `claude` в терминале |
| Правила | .cursorrules + .mdc | CLAUDE.md + agents/ |
| Автономность | Нужно подтверждать | Может работать сам |
| Для кого | Все уровни | Продвинутые |

## Зачем это бизнес-ассистенту

- **Автоматизация**: Claude Code может сам найти баг, исправить и протестировать
- **Масштаб**: запускаешь несколько агентов параллельно
- **Интеграции**: подключается к Git, MCP-серверам, любым CLI-инструментам

## Когда переходить на Claude Code

- Ты уверенно работаешь в Cursor
- Тебе нужна полная автоматизация (агент работает без тебя)
- Ты готов работать в терминале
- У тебя есть API-ключ Anthropic

Не торопись — Cursor покрывает 90% задач бизнес-ассистента. Claude Code — это следующий уровень.
