# AgentChattr — чат, где нейронки общаются между собой

**Источник:** https://github.com/bcurts/agentchattr

## Что это

Локальный чат (как Slack), в котором несколько AI-агентов общаются между собой и с тобой. Ты ставишь задачу — агенты сами координируются и решают её.

## Как работает

1. Запускаешь сервер на компе (один скрипт)
2. Открываешь браузер на `localhost:8300` — там чат
3. Запускаешь агентов (каждый в отдельном терминале)
4. Пишешь: `@claude какая архитектура лучше для этого бота?`
5. Claude отвечает и сам пишет: `@codex напиши базовую структуру`
6. Codex подхватывает, пишет код, спрашивает: `@gemini проверь на баги`
7. Цепочка крутится сама, пока не решат задачу

## Поддерживаемые агенты

- Claude Code
- Codex (OpenAI)
- Gemini CLI
- Kimi
- Qwen
- Kilo CLI
- MiniMax
- Любой MCP-совместимый агент

## Фичи

- **Каналы** — как в Slack (#general, #debug, #frontend)
- **Роли** — назначаешь агентам: Planner, Builder, Reviewer, Researcher
- **Задачи (Jobs)** — любое сообщение → задача с трекингом (To Do → Active → Closed)
- **Правила** — агенты сами предлагают правила работы, ты одобряешь
- **Защита от циклов** — после N ходов агенты останавливаются и ждут `/continue`
- **Голосовой ввод и картинки**

## Примеры использования

### Разработка бота
- Claude придумывает архитектуру
- Codex пишет код
- Gemini ревьюит и находит баги
- Claude фиксит
- Ты только читаешь чат и направляешь

### Исследование
- Gemini гуглит информацию
- Claude анализирует и пишет выводы
- Qwen проверяет факты

### Рефакторинг
- Один агент находит плохой код
- Второй переписывает
- Третий тестирует

## Как запустить

### macOS
```bash
# Установить tmux
brew install tmux

# Клонировать
git clone https://github.com/bcurts/agentchattr
cd agentchattr/macos-linux

# Запустить Claude
sh start_claude.sh

# Запустить Codex (в новом терминале)
sh start_codex.sh

# Открыть чат
open http://localhost:8300
```

### Windows
Открыть папку `windows/` и запустить:
- `start_claude.bat`
- `start_codex.bat`
- Открыть `http://localhost:8300`
