# Clawd Cursor — AI управляет компьютером как человек

**Источник:** https://github.com/AmrDab/clawd-cursor
**Сайт:** https://clawdcursor.com

## Что это

Скилл для OpenClaw / Claude Code / Cursor, который даёт AI-агенту «глаза и руки» — он видит экран, двигает мышку, кликает, вводит текст. Работает с любой программой, не только с кодом.

## Зачем

Вместо интеграции с 12 разными API (Gmail, Slack, Jira, Figma...) — один скилл, который работает через экран. Если ты можешь кликнуть — агент тоже может.

- **0 API-ключей** — если ты залогинена, агент тоже
- **0 серверов** — всё работает локально
- **Любое приложение** — браузер, 1С, Excel, CRM, госуслуги

## Как работает (3 уровня)

### Уровень 1: Action Router (мгновенно, бесплатно)
- Простые задачи: открой приложение, кликни кнопку
- Без нейронки — паттерн-матчинг + accessibility tree
- ~2 секунды, $0

### Уровень 2: Accessibility Reasoner (быстро, дёшево)
- Средние задачи: заполни форму, найди элемент
- Текстовая модель анализирует UI-дерево (без скриншотов)
- ~1 секунда, бесплатно с Ollama

### Уровень 3: Screenshot + Vision (надёжно)
- Сложные задачи: многошаговые сценарии в незнакомых интерфейсах
- Скриншот → vision-модель → план → выполнение → проверка
- 100% надёжность на тестах

## Практические применения

**Для ассистента:**
- Заполнить 20 карточек товара на маркетплейсе
- Собрать данные из CRM в таблицу
- Скачать счета из интернет-банка
- Заполнить формы на госсайтах

**Для бизнеса:**
- Рутина в программах без API (1С, AmoCRM, Bitrix)
- Тестирование сайтов — агент проходит путь клиента
- Автоматизация отчётов из нескольких систем

## Безопасность

3 уровня безопасности:
- **Auto** — открытие приложений, навигация (без подтверждения)
- **Preview** — ввод текста, заполнение форм (логируется)
- **Confirm** — отправка сообщений, удаление файлов (ждёт одобрения)

## Установка

```bash
git clone https://github.com/AmrDab/clawd-cursor.git
cd clawd-cursor
npm install && npm run setup
clawdcursor doctor
```

Требования: Node.js 20+
