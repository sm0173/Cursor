# Майнор — исходные материалы

Сюда кладите файлы курса: **pdf**, **docx**, **pptx**, старые **ppt**, при необходимости **sql**. Можно и в подпапки — скрипт обходит рекурсивно.

Для **.ppt** без LibreOffice в системе получится только короткая заглушка в `.md`; надёжнее сохранить слайды как PDF и положить сюда.

Потом из корня репозитория выполните:

```text
python Прога/scripts/convert_minor_course_materials.py
```

Появится папка **`Майнор - конвертированное`** с `.md` и копиями `.sql`.

Общая логика конвертации вынесена в [`Прога/scripts/course_materials_convert.py`](../Прога/scripts/course_materials_convert.py) (её же использует скрипт для курса «Базы данных»).
