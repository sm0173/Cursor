# Базы данных — конвертированные материалы

Здесь лежат выгрузки в **Markdown** (`.md`) из PDF и DOCX из папки `Базы данных/` и копии файлов **.sql** (если были в исходниках).

Формат `.md` удобен для поиска и для контекста в ИИ. Сложные слайды в PDF могут дать мало текста (без OCR).

Пересобрать выгрузку:

```text
python Прога/scripts/convert_bd_course_materials.py
```

Обработано в последний запуск: PDF — 5, DOCX — 8, SQL — 1.
