ЗАДАНИЕ №4.


к групповому проекту по учебной дисциплине «Базы данных»


Реализация простой бизнес-логики


Реализуйте заполнение базы данных тестовыми данными, в объеме достаточном для тестирования корректности проверки заданий на реализацию бизнес-логики.

Реализуйте основную бизнес-логику (запросы, представления, пользовательские функции) в соответствие с ранее сформулированными функциональными требованиями. Протестируйте.


В результате выполнения задания, должны быть приложены:

- выгрузку БД (backup);

- скрипты DML (файл .sql).


Пояснения к заданию.


Для выполнения п.1 этого задания необходимо написать запросы на вставку, либо пользовательские функции, в теле которых выполняются запросы на вставку. Функции могут принимать все входные данные в качестве параметров, либо осуществлять заполнение таблиц данными, генерируемыми автоматически в соответствии с имеющимися ограничениями.

Реализация бизнес-логики выполняется на языке SQL. Для упрощения запросов можно использовать представления, общие табличные выражения, вложенные запросы, вызов одних функций внутри других.

Примеры реализации функциональных требований.

F1. Бронирование переговорных комнат

F1.1 - вывод сведений о занятости переговорных комнат на определенную дату

Пример результата

Дата: 2023-03-01

| Комната | Время начала встречи | Время окончания встречи |
| --- | --- | --- |
| 3 | 15:00:00 | 16:00:00 |
| 3 | 17:00:00 | 18:00:00 |
| 10 | 09:00:00 | 10:45:00 |
| … |  |  |


F1.2 - вывод сведений о свободных на выбранную дату и период времени переговорных комнатах

Пример результата

Дата: 2023-03-01

Период времени: 09:00:00 – 11:00:00

| Комната | Вместимость, человек | Этаж |
| --- | --- | --- |
| 5 | 5 | 1 |
| 7 | 10 | 2 |
| … |  |  |
