## Страница 4

Структура языка SQL
Structured Query Language (SQL) — язык структурированных
запросов
Data Definition Language (DDL) – это группа операторов языка SQL для
определения структуры данных
Data Manipulation Language (DML) – это группа операторов языка SQL для
манипулирования данными (добавление, изменение, удаление, …)
Data Control Language (DCL) – это группа операторов языка SQL для
контроля доступа к данным
Transaction Control Language (TCL) – это группа операторов языка SQL для
работы с транзакциями
