# Карта репозитория (живой документ)

Обновляется агентом структуры и вручную при крупных изменениях. Цель — чтобы любой (и ИИ) быстро понимал, **где что лежит**.

| Зона | Назначение |
|------|------------|
| `.cursor/rules/` | Правила Cursor: роли, ограничения по папкам |
| `docs/` | Документация, публикации (например GitHub Pages из `docs/`) |
| `tasks/` | Планы, todo, уроки (`lessons.md`), **карта и память структуры** |
| `HTML сайта продакта/` | Статические страницы учебного/визиточного сайта |
| `Курсач по моделькам/` | Курсовая «Моделирование организации», шаблоны глав и материалы |
| `Инструкции курса/` | Материалы обучения Cursor/агентам: презентации, шаблоны `.mdc`, вложенные репозитории-примеры |
| `Прога/` | Эталон: корень — `README.md`; [`docs/`](../Прога/docs/) — каталог ([`INDEX.md`](../Прога/docs/INDEX.md)); [`scripts/`](../Прога/scripts/) — конвертация; **`01_`…`03_`** — материалы; для ИИ — `.txt` в `02_`/`03_`, не оригиналы |

## Индекс зоны `Прога/` (зафиксировано для агента структуры)

Полный перечень имён по папкам — в [`Прога/docs/INDEX.md`](../Прога/docs/INDEX.md).

**Слой документации и кода**

| Путь | Назначение |
|------|------------|
| `Прога/README.md` | Точка входа: правила контекста, запуск конвертации |
| `Прога/docs/README.md` | Что лежит в `docs/` |
| `Прога/docs/INDEX.md` | Дерево зоны и таблицы файлов |
| `Прога/scripts/convert_materials.py` | Оригиналы → `.txt` |
| `Прога/scripts/requirements-materials.txt` | Зависимости для конвертации |

**Папки с материалами**

| Папка | Назначение |
|-------|------------|
| `Прога/01_Оригиналы материалов/` | Исходные `.ipynb` / `.pdf`; **не** основной контекст для ответов ИИ |
| `Прога/02_Материалы по ИАДу/` | Выгрузки **`Sem …`** (ИАД) |
| `Прога/03_Материалы по питону/` | Выгрузки **`Seminar…`** (Python и смежное) |
