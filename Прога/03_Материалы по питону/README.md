# Материалы по питону

Текстовые выгрузки семинаров **«Seminar…»** из папки `01_Оригиналы материалов` (курс по Python и смежным темам).

Формат такой же, как в соседней папке `02_Материалы по ИАДу`: `.ipynb` → `.txt` через `convert_materials.py`. Семинары ИАД (`Sem …`) складываются в `02_Материалы по ИАДу`, семинары Python (`Seminar…`) — сюда.

## Как обновить выгрузку

Из корня репозитория (или из папки `Прога`):

```bash
pip install -r "Прога/scripts/requirements-materials.txt"
python "Прога/scripts/convert_materials.py"
```

После добавления новых `.ipynb` в `01_Оригиналы материалов` снова запустите скрипт — соответствующие `.txt` перезапишутся в нужной папке.
