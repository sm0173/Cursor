# Материалы по ИАДу

Текстовые выгрузки из папки `01_Оригиналы материалов` для семинаров **«Sem …»** (интеллектуальный анализ данных). Семинары **«Seminar…»** (Python и смежные темы) лежат отдельно: [`../03_Материалы по питону/`](../03_Материалы%20по%20питону/).

- **`.ipynb`** → `.txt` (Markdown через nbconvert: заголовки, код, текст; картинки в ноутбуках — длинные base64).
- **`.pdf`** → `.txt` (текст страниц через PyMuPDF; сканы без слоя текста не распознаются — нужен отдельный OCR).

## Как обновить выгрузку

Из корня репозитория (или из папки `Прога`):

```bash
pip install -r "Прога/scripts/requirements-materials.txt"
python "Прога/scripts/convert_materials.py"
```

Только ноутбуки или только PDF:

```bash
python "Прога/scripts/convert_materials.py" --ipynb-only
python "Прога/scripts/convert_materials.py" --pdf-only
```

После добавления новых файлов в `01_Оригиналы материалов` снова запустите `convert_materials.py` — соответствующие `.txt` перезапишутся в этой папке или в `03_Материалы по питону` в зависимости от префикса имени (`Sem …` / `Seminar…`).
