# Каталог зоны `Прога/`

**Назначение:** снимок **папок и файлов** зоны `Прога`: что в корне, в `docs/`, `scripts/` и в трёх папках материалов. Правила контекста для ИИ — в [`README.md`](../README.md) в корне `Прога/`.

**Префиксы `01_` … `03_`** у папок с материалами — порядок в проводнике: исходники → ИАД → Python.

**Обновление:** после добавления или удаления файлов поправь списки ниже.

---

## Дерево (эталонная структура)

```
Прога/
├── README.md
├── docs/
│   ├── README.md
│   └── INDEX.md          ← этот файл
├── scripts/
│   ├── convert_materials.py
│   └── requirements-materials.txt
├── 01_Оригиналы материалов/
├── 02_Материалы по ИАДу/
└── 03_Материалы по питону/
```

Идея как в шаблонах курса: **документация отдельно** (`docs/`), **автоматизация отдельно** (`scripts/`), **данные/материалы** в своих папках.
