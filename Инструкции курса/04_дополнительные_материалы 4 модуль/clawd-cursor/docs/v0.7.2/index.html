Eight Layers

OCR first, vision last

Each task falls to the cheapest layer that can handle it. OCR reads the screen as text. Vision is the absolute last resort. Only active in CLI agent mode.

🌐

L0 · Browser / CDP — $0

Direct DOM interaction via Chrome DevTools Protocol. Click by selector, read text, navigate. No screenshots.

⚡

L1 · Action Router — $0

Pattern-matched tasks + 30 keyboard shortcuts. Open apps, type text, navigate URLs. Zero LLM calls.

🔧

L1.5 · Deterministic Flows — $0

Hardcoded sequences for common tasks: email compose, find & replace, app switching. Zero LLM, instant.

🗂️

L2 · Skill Cache — $0

Replays learned action patterns. Skips perception entirely for previously-solved interactions.

👁️

L2.5 · OCR Reasoner ★ — text model

PRIMARY layer. OS-level OCR reads screen as text. Cheap text LLM reasons per step. Stagnation detection + done-verification.

♿

L2.5b · A11y Reasoner — text model

Fallback when OCR is unavailable. Accessibility tree + text LLM. Circuit breaker on repeated failures.

🎯

L3 · Computer Use — vision model

Last resort. Full vision model with screenshot loop. Supports Anthropic native + any OpenAI-compatible vision model.

🔍

Pre-processor

Regex parser runs first (zero cost), then 1 cheap LLM call decomposes compound tasks into structured intent.

What's New

Universal. Hardened. 13 providers.

v0.7.2 adds security hardening, 13 auto-detected providers, DPI-safe coordinates, and install verification. OCR reads the screen as text — vision is the last resort.

👁️

OCR-First Pipeline

Windows.Media.Ocr reads every pixel on screen as text with bounding boxes. A11y tree runs in parallel as a supplement. Zero screenshots for most tasks.

🖱️

Smart Tools (Blind Agent)

4 new MCP tools: smart_read, smart_click, smart_type, invoke_element. Click buttons by name, not coordinates. No vision needed.

⌨️

Shortcuts Engine

Built-in keyboard shortcut database. shortcuts_list discovers available shortcuts. shortcuts_execute fires them instantly.

✅

Ground-Truth Verifier

Reads actual screen state after every action. Blocks false "done" reports — the agent can't lie about success anymore.

🔌

Native MCP Support

Plug directly into Claude Code, Cursor, Windsurf, or Zed as an MCP server. No REST plumbing — tools show up natively in your AI's registry.

🗂️

Skill Cache

Learns action patterns over time. Once it knows "new tab = ctrl+t", it skips all perception layers and executes instantly. Zero cost.

Get Started

Up and running in 2 minutes

Pick your OS, install, consent once, connect your AI.

PowerShell
# 1. Clone and install (run ALL on one line)
git clone https://github.com/AmrDab/clawdcursor.git --branch v0.7.2; cd clawdcursor; npm install; npm run setup

# 2. Auto-detect your AI providers
clawdcursor doctor

# 3. Start the agent (consent prompt on first run)
clawdcursor start

# 4. Send a task
clawdcursor task "Open Chrome and go to github.com"

# Or use as MCP server (add to Claude Code / Cursor config)
{
  "mcpServers": {
    "clawdcursor": { "command": "clawdcursor", "args": ["mcp"] }
  }
}
